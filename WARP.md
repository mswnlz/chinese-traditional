# Warp AI 优化配置

## 项目简介
这是一个中华传统文化相关的资源仓库，主要包含传统文化知识、历史资料和文化传承内容。

## 常用命令
```bash
# 查看最近更新的文档
ls -lt *.md | head -5

# 搜索特定内容
grep -r "关键词" *.md
```

## Git 工作流
```bash
# 添加新文档
git add 202508.md
git commit -m "Add 202508 traditional culture resources"
git push origin main
```

## 文档约定
- 文档命名格式：YYYYMM.md
- 内容包含传统文化、历史知识、文化传承
